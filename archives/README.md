# Archives

这里用于存放和关联我的历史学习材料、旧路线图、旧作业和早期探索项目。

这个目录不是垃圾箱，而是历史层。
旧内容会保留，但不再作为当前主线展示。

## 归档原则

* 当前主线内容放在 `projects/`、`notes/`、`logs/`
* 旧计划、旧作业、原始记录放在 `archives/`
* 早期仓库可以通过链接关联，不一定复制内容
* 归档内容保留学习痕迹，但不干扰当前结构
