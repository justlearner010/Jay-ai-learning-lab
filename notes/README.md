## 维护方式

1. 在 `notes/library/` 中编辑笔记。
2. 在子模块仓库中提交并推送笔记。
3. 回到 Learning Lab，提交更新后的子模块指针。

克隆或初始化笔记内容：

```bash
git submodule update --init --recursive
```
