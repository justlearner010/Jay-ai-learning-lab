
### 知识点复习

我发现自己在学习新知识的时候容易忘记对旧知识的复习，现在设立一个文件夹，通过codex的自动化功能帮助实现知识点复习的自动化，具体的复习规则是：
**每日早上9点检测新添加的笔记，并按新知识点 : 旧知识点 = 2 : 1 默认生成 3 个完整复习卡；如当天未复习新知识点很多，可扩展到 6 个并保持 4 : 2。**

- 源目录是 `notes/library`，输出目录是 `knowledge-review/content/YYYY-MM-DD/`。
- 每张卡保留 YAML front matter，并包含：知识点与来源、盲测题、我的作答区、折叠参考答案、复习结果、下次复习建议。
- 历史 `pending` 文件只有在用户明确勾选自评后才更新 `status` 与 `next_review_date`，不根据部分作答推测表现。
- 每日同时生成或更新 `YYYY-MM-DD-复习清单.md`，明确记录实际比例、安排原因和待完成状态。
