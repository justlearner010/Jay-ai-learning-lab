- 参考答案：给定一个数组，只要有一个数字出现至少两次就返回 True，否则返回 False

### 第 2 题

- 参考答案：这份笔记强调的信号是：需要在遍历过程中快速判断“另一个值是否已经出现”，所以用Set 做查询，避免暴力双循环。题目只关心元素是否重复，不需要保存下标或计数，因此使用 Set 保存已经出现的值
- 常见误区：把相近概念、步骤或角色混在一起。

### 第 3 题

- 参考答案：创建空 Set。；遍历数组；如果数字已经在 Set 中，立即返回 `True`。；否则加入 Set；遍历结束后返回 `False`。
- 常见误区：背了结论，但不会落到实际调用或流程里。
